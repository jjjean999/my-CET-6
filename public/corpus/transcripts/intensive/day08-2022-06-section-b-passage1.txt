have identified a potent new antibiotic compound using artificial intelligence. // 研究人员利用人工智能识别出了一种强效的新型抗生素化合物。
The antibiotic can kill very dangerous bacteria. // 这种抗生素可以杀死非常危险的细菌。
According to a study published in the journal Cell, the compound successfully removed deadly strains of bacteria in mice which are resistant to all known antibiotics. // 根据发表在《细胞》杂志上的一项研究，该化合物成功清除了小鼠体内对所有已知抗生素具有抗药性的致命菌株。
The researchers say this is the first time that artificial intelligence has been used to find a powerful new antibiotic molecule. // 研究人员表示，这是首次利用人工智能发现一种强大的新抗生素分子。
Why does this matter? // 为什么这很重要？
The answer is antibiotic resistance. // 答案是抗生素耐药性。
This happens when bacteria develop the ability to survive the medications designed to kill them. // 当细菌进化出能够在旨在杀灭它们的药物中生存的能力时，就会出现这种情况。
Antibiotic resistance is a serious threat to health and the problem is growing. // 抗生素耐药性对健康构成严重威胁，而且问题日益严重。
This makes finding new antibiotics very important. // 这使得寻找新型抗生素变得非常重要。
However, in recent decades, very few have been developed, and those that have tend to be very similar to drugs already available. // 然而，近几十年来开发出的新抗生素很少，而且已有的新药往往与现有药物非常相似。
The searches also tend to only focus on a narrow spectrum of chemical compounds. // 搜索范围也往往只集中在狭窄的化学化合物谱系中。
But this is where artificial intelligence comes in. // 但这正是人工智能大显身手的地方。
Why? // 为什么？
To find new drugs, scientists screen molecules to predict how effective they might be. // 为了发现新药物，科学家们筛选分子以预测其可能的效果如何。
Typically, such screening is done by humans in the lab, which is both costly and slow. // 通常，这类筛选由人类在实验室中进行，既昂贵又缓慢。
Artificial intelligence is different. It's fast, and it can process a high volume. // 人工智能则不同。它速度快，能够处理海量数据。
It can screen hundreds of millions of compounds to identify a few interesting candidates that require experimental testing. // 它可以筛选数亿种化合物，找出少数几个值得进行实验测试的有潜力的候选药物。
Artificial intelligence is also able to predict if compounds are likely to be toxic. // 人工智能还能够预测化合物是否可能有毒性。
Some experts assert that this work signifies a paradigm shift in antibiotic discovery. // 一些专家断言，这项工作标志着抗生素发现领域的范式转变。
It could change drug discovery more generally. // 它可能会更广泛地改变药物研发的方式。
